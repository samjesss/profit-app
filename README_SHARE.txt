# Cómo usar Profit (Para tu amigo)

¡Hola! Para usar la App Profit en tu computadora, sigue estos pasos sencillos.

## 1. Requisitos Previos
Necesitas tener **Python** instalado.
- Descárgalo aquí si no lo tienes: https://www.python.org/downloads/
- Al instalar, marca la casilla: **"Add Python to PATH"**.

## 2. Instalación (Solo la primera vez)
1. Abre la carpeta `Profit`.
2. Haz doble clic en el archivo `install_requirements.bat` (si no existe, crea uno o abre una terminal y escribe: `pip install -r backend/requirements.txt`).

## 3. Ejecutar la App
1. Haz doble clic en **`run_webapp.bat`**.
2. ¡Listo! Se abrirá en tu navegador.

---
**Nota:** Como la base de datos está en la nube (Supabase), ambos verán los mismos datos actualizados en tiempo real.
